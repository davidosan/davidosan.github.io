Certaines sessions peuvent être particulièrement efficaces pour lutter contre des douleurs ou inconforts liés à la fibromyalgie :

  • Une session le matin pour les raideurs matinales
  • Une session en milieu ou fin d’après midi pour réduire la remontée des douleurs
  • Une session avant ou après un effort physique pour réduire les douleurs, les courbatures,
  • Une session durant une station immobile prolongée pour réduire la raideur et les douleurs associées,
  • Une session dans l’heure précédant le coucher pour faciliter l’endormissement,
  • Une session lors de réveils nocturnes, afin de vous aider à vous rendormir.
